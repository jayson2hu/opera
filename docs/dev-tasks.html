Codex 开发指引

开发流程

  • 按 T-01 → T-11 顺序逐条开发
  • 每完成一个任务，运行 npm run lint 和 npm run build 验证
  • 验证通过后，使用对应的 commit message 提交
  • 提交后将本文档中对应任务的状态从 s-todo 改为 s-done

项目路径

  • 前端代码: opera-app/src/
  • 组件目录: opera-app/src/components/
  • 页面目录: opera-app/src/pages/
  • 类型定义: opera-app/src/types.ts
  • 常量定义: opera-app/src/constants.ts

验证命令

  • Lint: cd opera-app && npm run lint
  • Build: cd opera-app && npm run build
  • Dev server: cd opera-app && npm run dev

提交规范

  • 每个任务一个 commit，使用任务卡片中指定的 commit message
  • 不要合并多个任务到一个 commit
  • 提交前确保 lint 和 build 都通过
